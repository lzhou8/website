Microaggressions ML

Introduction:

Microaggressions are verbal statements with implicit negative intent, especially towards those of marginalized communities. Although there are plenty of hate speech classifiers on the Internet, I was unable to find many machine learning endeavors that were able to classify microaggressions, which often convey indigities in a subtler form. Due to this reason, I was inspired to create a dataset of my own and experiment with different text classification models to see if they could accurately classify text as a microaggression or non-microaggression.

Completed Work:

Due to the lack of pre-existing datasets online, I decided to create my own dataset with both microaggressions and non-microaggressions. I found a website called www.microaggressions.com and wrote a Python script (leveraging pytumblr) to pull posts containing microaggressions from the website. I also downloaded a dataset of common English sentences to be used as non-microaggressions. With both data representing micro-aggressions and non-microaggressions, I aggregated the data into one CSV file.

With the CSV file, I created a Jupyter notebook to preprocess text (which included removing stop words and punctuation) and used this preprocessed text to train several different text classification models. These models were Bernoulli Naive Bayes (achieving an accuracy of 83%), Logistic Regression (achieving an accuracy of 80%), and a Support Vector Machine (achieving an accuracy of 83%). Using these trained models, I was able to use them to predict on new data.

Conclusion:

Achieving high accuracy with machine learning models requires lots of data, as well as data that represents their labels well. In the future, this initial work could be expanded upon by training models to recognize the specific type of microaggression. As of now, the machine learning models with this project have performed with a highest accuracy of 83% on a dataset of almost 3,000 microaggressions and non-microaggressions combined.
