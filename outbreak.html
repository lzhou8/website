Georgia Tech HealthTech Hackathon: Disease Symptoms Reporting Application

Introduction:

With 2 students, I developed an Android application that would enable patients to easily connect with their physicians. As a proof of concept, the application allows users to login, create a report with their symptoms and current medications, and send the report via email or SMS messaging to their local healthcare provider.

Technologies:

I used Java for back-end and XML for the front-end. For more information, the full project description can be viewed here.
